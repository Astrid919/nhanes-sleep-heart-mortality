# Data Availability Statement

The cleaned analysis dataset, analysis scripts, output tables, and figure-generating code are available at GitHub: https://github.com/Astrid919/nhanes-sleep-heart-mortality. The original NHANES public-use data are available from NCHS NHANES at https://wwwn.cdc.gov/nchs/nhanes/, and public-use linked mortality files are available from NCHS linked mortality resources at https://www.cdc.gov/nchs/data-linkage/mortality-public.htm.

For long-term archival, a versioned GitHub release with a persistent Zenodo DOI is recommended before final submission.
